# Amatsubame

A toy web browser implemented in Swift, following [Web Browser Engineering](https://browser.engineering/) by Pavel Panchekha & Chris Harrelson.

## Build

```bash
swift build
```

## Usage

```bash
.build/debug/Amatsubame <url>
```

```bash
.build/debug/Amatsubame http://example.org/
```

## Requirements

- macOS 12+
- Swift 5.9+
